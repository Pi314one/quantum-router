# 说明
## gjq(国基量子云)平台使用
本项目支持使用python调用gjq平台的服务,由gjq-client库提供,具体使用方法请参考国基量子云平台官方教程 https://www.tiangongqs.com/docs/zh/quantum/QuantumProgrammingFramework.html ， `...\scripts\example.py`是Sampler任务的使用示例，`...\scripts\example1.py`是Estimator的使用示例。api_key存放在文件`gjq-api-key.txt`中。

## KaiWu SDK使用
kaiwu SDK的使用说明详见官方教程 https://kaiwu-sdk-docs.qboson.com/zh/v1.3.1/index.html ，`...\scripts\example2.py`是使用kaiwu SDK调用真机求解TSP问题的示例。许可证存放在文件`kaiwu-sdk-license.txt`中。
